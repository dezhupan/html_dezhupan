1. 线性回归拟合线：标准公式 ŷ = wx + b

本组件只使用标准公式：ŷ = wx + b。其中 w 或 k 表示斜率，b 表示截距/偏置，也就是 x=0 时预测线和数学 y 轴的交点。

重点：身高数据的 x 大约在 150~190。如果只看局部区间，单独拖动 w 时，红线会同时改变高度和倾斜程度。为了让初学者看清楚“w 控制斜率”，这里默认开启“拖动 w 时固定当前预测点”，也就是拖动 w 时自动换算 b，让红线围绕黑点旋转。

为什么 b 常常是负数？

因为 b 是 x=0 时的预测值，而身高数据集中在 150~190cm 附近。比如 ŷ = 0.9x - 88，当 x=176 时预测约 70.4kg，但当 x=0 时就是 -88。这只是直线向左延长后的数学截距，不表示真实的 0cm 人体重。

2. MAE / MSE / RMSE 自动计算器

MAE：平均绝对误差先取绝对值再平均。人话：平均错多少。
MSE：均方误差先平方再平均。人话：大错误会被重罚。
RMSE：均方根误差MSE 开平方。人话：回到原单位，更好解释。
三个指标怎么选？

MAE 最好解释，适合说“平均差多少”。MSE 对大误差更敏感，适合重点惩罚离谱预测。RMSE 也惩罚大误差，但单位和原目标值一致，比 MSE 更容易读。

3. 导数：看当前点的变化趋势

函数：y = x²，导数：y' = 2x。红线是当前位置的切线，切线斜率就是这一点的导数。

导数 > 0：往右走函数值上升，上坡。 导数 < 0：往右走函数值下降，下坡。 导数 = 0：切线水平，可能到山底或山顶。

4. 梯度下降：沿着“负梯度”一步一步下山

简化损失函数：J(w) = (w - 3)²。这里的 w 是模型要学习的权重参数，可以先理解成直线公式 ŷ = wx + b 里的斜率/权重。为了演示梯度下降，本组件先只优化一个参数 w。

先记一句：w 是模型参数；J(w) 是这个参数带来的损失；梯度表示损失上升最快的方向。我们要让损失下降，所以参数更新要减去梯度：w新 = w旧 - 学习率 × 梯度。
w 是什么：w 是模型权重参数。在线性回归 ŷ = wx + b 中，w 决定特征 x 对预测值的影响力度。 损失：J(w) = (w - 3)²，表示当前 w 错得多严重；越接近 0 越好。 梯度：grad = J'(w) = 2(w - 3)，表示当前点损失上升最快的方向。 更新：w_new = w_old - α × grad，减号表示沿“负梯度方向”下山。
为什么原来的图初学者看不懂？

只画一条曲线和一条红线太抽象。初学者需要同时看到：当前位置、山底、梯度方向、负梯度方向、学习率控制步长、每一步更新后的损失变化。这个版本把这些信息全部标在图上。

5. 标准化和数据泄露判断

标准化公式：z = (x - mean) / std。机器学习里，训练集可以 fit_transform，测试集只能 transform。

为什么测试集不能 fit_transform？

测试集相当于考试题。fit 会学习均值和标准差，如果对测试集也 fit_transform，就等于提前看了考试题的统计信息，评估结果会不真实，这叫数据泄露。
